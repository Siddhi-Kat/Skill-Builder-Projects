# 💳 AtliQo Bank Credit Card Launch Analysis

## 📌 Project Overview
This project analyzes customer transaction behavior and credit profiles to support the launch of the **AtliQo Bank Credit Card**.

The project focuses on:

- identifying the most suitable customer segment for the credit card launch
- evaluating campaign effectiveness using statistical testing

---

## 📂 Dataset
The analysis was performed using the following datasets:

- `customers.csv`
- `credit_profiles.csv`
- `transactions.csv`
- `avg_transactions_after_campaign.csv`

These datasets include customer demographics, transaction activity, payment behavior, and credit-related attributes.

---

## 🚀 Project Phases

### Phase 1 — Customer Segmentation
Customer, transaction, and credit profile data were analyzed to identify high-potential customer groups.

### Phase 2 — Campaign Evaluation
A post-campaign statistical test was conducted to determine whether the campaign led to a significant increase in customer transactions.

---

## 📊 Key Insights
Key business insights and visual analysis are included in the **presentation deck (PDF)** attached in this repository.

---

## 🎯 Business Outcome
The analysis suggests that **customers aged 26–48 should be prioritized for the initial AtliQo credit card launch**, while the campaign results indicate a **significant positive impact on transaction activity**.
